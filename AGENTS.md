# Access Todo — 에이전트 지시 규칙

이 프로젝트는 Access 포스트잇 TODO 시스템과 연동되어 있습니다.
`access-todo` MCP 서버가 ZCode에 등록되어 있습니다.

## 🔴 필수 규칙: 작업 완료 시 자동 체크

사용자가 요청한 작업을 완료한 후 (코드 수정, 버그 수정, 문서 작성 등):

1. `search_todos` (agent: "zcode") 로 관련 할 일 검색
2. 해당 할 일의 id를 찾으면 `toggle_todo` (agent: "zcode") 로 체크
3. "✓ Access 포스트잇에서 체크했습니다" 라고 사용자에게 알림

**체크 조건:**
- 사용자가 "완료", "끝", "done", "고쳤어" 등 완료 의미 표현
- 실제 코드/파일 변경 완료

**체크하지 않는 조건:**
- 단순 질문/조회
- 작업 미완료
- Access에 해당 할 일이 없음 → 새 추가 여부를 사용자에게 확인

## 새 작업 발견 시

버그 발견, 새 기능 필요 등의 상황에서:
- `add_todo` (agent: "zcode", tags: ["agent:zcode"]) 로 자동 추가
- 우선순위는 상황에 맞게 판단 (긴급 버그 = high)
